# 2026.06.06 最新建站企业站点导航网

这是一个基于 Web 技术构建的高性能、极速响应的企业级建站导航门户站点。

## 功能特点

- **极速检索**：支持对 1550 个企业站点及域名进行即时本地拼音/文字检索与过滤。
- **现代化设计**：深色模式背景，搭配渐变霓虹与毛玻璃质感卡片设计。
- **SEO 友好**：静态 HTML 渲染，利于搜索引擎蜘蛛抓取和权重传递。

## 如何通过 GitHub Pages 免费上线发布

您只需将本仓库推送到 GitHub，即可一键上线：

1. **新建 GitHub 仓库**：
   - 登录 [GitHub](https://github.com/)。
   - 新建一个公共仓库（Public），命名为例如 `enterprise-sites-2026`。

2. **关联本地仓库并推送**：
   - 我们已经帮您初始化了本地 git 仓库。您只需执行：
     ```bash
     git remote add origin https://github.com/您的用户名/您的仓库名.git
     git branch -M main
     git push -u origin main
     ```

3. **开启 GitHub Pages**：
   - 进入 GitHub 仓库页面，点击右上角的 **Settings** (设置)。
   - 在左侧菜单栏选择 **Pages**。
   - 在 **Build and deployment** 下的 **Source** 选择 `Deploy from a branch`。
   - **Branch** 选择 `main`，目录选择 `/ (root)`，点击 **Save**。
   - 等待约 1 分钟，GitHub 会在页面上方提供一个绿色的链接（例如 `https://您的用户名.github.io/您的仓库名/`），点击该链接即可在线访问您的导航主页，且所有外链均可正常点击和跳转！
